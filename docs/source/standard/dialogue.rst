.. _Dialogue:

Dialogue
========

Schema
------

:title:
   string, required

:description:
   string, required

:answer:
   string

:documents:
   List of :ref:`document` objects

:dateSubmited:
   string, :ref:`date`, autogenerated

:dateAnswered:
   string, :ref:`date`, autogenerated

:dialogueOf:
    string, autogenerated

    Possible values for :ref:`decision` or :ref:`conclusion` are:

    * `decision`
    * `conclusion`


:relatedParty:
    string

    Id of related :ref:`party`.

:author:
    string, autogenerated

    Possible values are:

    * `monitoring_owner`
    * `tender_owner`
